## 抖音爆款开场白公式

### 情绪共鸣型
- "家人们，谁懂啊！"
- "破防了家人们！"
- "真的绷不住了！"

### 痛点唤醒型
- "别划走！这个你一定要看"
- "90%的人都不知道..."
- "你是不是也经常这样？"

### 反差吸引型
- "后悔没早系列+1"
- "居然可以这样？"
- "刷新我的认知！"

- "打工人/学生党必看！"
- "数码党狂喜！"
- "租房党/有车党必看！"

## 镜头/画面建议公式

每条文案后添加【图片/镜头建议】模块：

```
【图片/镜头建议】
[0-3秒] 开场画面：痛点/情绪/悬念相关
[3-10秒] 痛点/场景图：引发共鸣的场景照片
[10-20秒] 产品图：实物展示（正面/侧面/细节）
[20-30秒] 使用场景：真实使用场景照片/视频
[30秒后] 效果/对比：数据图/对比图/好评截图
```

### 常见镜头类型

| 镜头 | 适用场景 | 示例 |
|------|----------|------|
| 痛点画面 | 开场钩子 | 电量警告、接口损坏、线材缠绕 |
| 产品展示 | 产品介绍 | 实拍图、细节图、拆箱视频 |
| 使用场景 | 真实体验 | 通勤、办公、旅行、居家 |
| 对比实验 | 效果展示 | 速度对比、质量测试、性价比 |
| 数据图表 | 卖点强化 | 续航数据、充电速度、价格对比 |
| 好评截图 | 信任背书 | 用户评价、销量数据、排行榜 |
| 表情包/gif | 情绪强化 | 夸张表情、搞笑动图 |
