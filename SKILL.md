# 短视频文案一键生成器

一键批量生成适合抖音等社交媒体的短视频文案，支持产品种草、知识分享等多种风格。

## 默认配置

| 配置项 | 默认值 |
|--------|--------|
| 领域 | 数码产品 |
| 平台 | 抖音 |
| 风格 | 产品种草/推荐 |
| 数量 | 3条 |

用户可随时修改这些配置。

## 使用方法

### 一键生成（使用默认配置）

用户说：
- "一键生成短视频文案"
- "帮我生成3条抖音文案"
- "批量生成产品种草文案"

直接按默认配置生成，无需确认。

### 自定义生成

用户可指定参数：
- **领域**：数码产品、美妆、食品、服装等
- **平台**：抖音、小红书、视频号、B站
- **风格**：产品种草、知识干货、情感故事、热点评论
- **数量**：1-10条

示例：
- "帮我生成5条小红书美妆种草文案"
- "生成3条B站数码产品评测脚本"

## 文案模板结构

每条文案包含：

```
【标题】吸睛开场（5-15字）

【开场钩子】
前3秒抓住注意力的开场白（20-30字）

【痛点/场景】
描述用户痛点或使用场景（3-5句，每句8-15字）

【产品介绍】
自然引入产品，突出核心卖点（4-6句，每句8-15字）

【真实体验/效果】
个人使用感受或效果展示（3-5句）

【行动引导】
引导点赞、关注、评论或购买（1-2句）

【Hashtag】
5-8个相关话题标签
```

## 输出格式

生成后以列表形式展示：

---

**文案 1**

标题：xxx

[开场钩子]
...

[完整文案内容，每句单独一行]

#标签1 #标签2 #标签3

---

## 质量标准

1. **口语化**：适合短视频口播，避免书面语
2. **有情绪**：戳中用户痛点或爽点，引发共鸣
3. **有节奏**：句子短促有力，适合快节奏视频
4. **有画面感**：描述具体场景，让用户有代入感
5. **有细节**：加入真实使用感受和具体数据
6. **有引导**：结尾有明确的互动或转化引导
7. **平台适配**：符合抖音的算法喜好和用户习惯
8. **长度适中**：每条文案80-150字，确保口播时长30-60秒
